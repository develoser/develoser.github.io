  • Handlebars precompiler (Sublime Text 2 Plugin) ,  GitHub

    Published on: Jan 25, 2013

    I really like Sublime Text 2 as IDE and before we migrated to RequireJS and hbs loader we used to precompile the handlebars templates after any modification over and over again, it includes:

    • Open shell (terminal, cmd, PowerShell, etc.)
    • Change directory to the template path
    • Type the compile commands
    • Open the generated file, and so on… boring!

    I looked for a nice Sublime Text 2 plugin. Bad luck, I did not found anyone, so I decided to write my own (Python + GitHub) and avoid wasting time in console, just right click and select “Precompile with Handlebars” and that’s it. Faster, better ;)
